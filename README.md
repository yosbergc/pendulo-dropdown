# The context menu alternative for React/Next.js

Sometimes we just want to add a custom context menu to our projects, but then we find we will need to implement a great amount of code that will take us some time to create, even with this AI era, those are some tokens we dont want to waste as the main point should always be the final product. Thats why I wanted to make a solution for the people like me who use React/Next.js as their main framework for frontend.

## Documentation
[Documentation](https://pendulo-dropdown.vercel.app/)
[NPM Link](https://www.npmjs.com/package/pendulo-dropdown)

## Features

- Light/dark theme.
- Keyboard navigation.
- Keyboard shortcuts for specific items.
- Hide or disable items dinamically.
- Automatically responsive
