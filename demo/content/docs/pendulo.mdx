---
title: Pendulo
description: Context menu component.
---

The pendulo component is the main point of this library, it's the container that will have all the logic and components that you will render on all your menus/context menu. You can send it a darkMode prop that will allow you to use the dark mode of the library.

| Property      | Description                                                           | Accepted Type
| ----------    | --------                                                              | --------
| darkMode?    | Define the theme the library will use for the dropdown/items style.   | `boolean` |
